Projetos

Projeto Dr. Strange 2

Este projeto foi realizado sobre a orientação do Professor responsável durante o tempo de curso no SENAI de Jandira como um método de avaliação com o principal objetivo de criar um site propaganda para o novo filme da Marvel: Doutor Estranho no Multiverso da Loucura. Personagem

O projeto do personagem foi realizado durante o tempo de curso no SENAI de Jandira como um método de avaliação, tendo como principal objetivo de criar um site apresentando e descrevendo um personagem da nossa escolha, sendo ele de jogos, séries ou alguma pessoa real.

CEM

Este projeto foi realizado em grupos, juntando o curso de Desenvolvimento de Sistemas com o curso Técnico de Administração, utilizado como avaliação final do semestre e usando todo o nosso conhecimento acumulado no semestre sobre o HTML, o CSS e o FIGMA. Apresentado em forma de seminário, o objetivo era promover uma marca criada pelos estudantes do curso de administração com uma série de páginas web da empresa, mostrando com o que a empresa trabalha como uma forma de chamar a atenção para fechar as vagas de emprego que abriram com as novas localizações.
